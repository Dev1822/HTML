HTML Elements Showcase – Inline vs Block

Introduction: In HTML, elements are broadly divided into two categories: block-level and inline. Block elements take up the full width, while inline elements only take as much space as needed.

Block-level Elements | Inline Elements | Comparison Table

Block Elements

Paragraph

Example:

This is a paragraph

Explanation: The <p> tag defines a block of textthat always starts on a new line.


Heading

Example:

h1

h2

h3

h4

h5
h6
Explanation: Headings are block-level elementsthat take the full width.


Blockquote

Example:
"Coding is the new literacy."
Explanation: Blockquote is a block container for quotations.


HR

Example:
Explanation: hr is a block container for making a horizontal line.


Table

Example:
Roll No. Name
1 ABC
Explanation: table is a block container for making a table.


List

Example:
  • Item 1
  • Item 2
  1. Item 1
  2. Item 2
Explanation: ul and li (list) is a block container for making a list.


Form

Example:
Explanation: form is a block container for making a form.


div

Example:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Explicabo cupiditate consectetur commodi est sit obcaecati quisquam tempora veritatis nihil suscipit, ab harum mollitia nostrum veniam provident facere hic amet iusto.

Explanation: div is a block container for making a Generic container for grouping content.


header

Example:
Welcome
Explanation: header is a block container for representing a page or section header.


footer

Example:
Thank You
Explanation: footer is a block container for representing a page or section footer.

Inline Elements

Span

Example:

This is a blue inline span inside a sentence.

Explanation: The <span> tag is an inline container used to style part of a text without breaking the flow.


Anchor

Example:

Visit Example for more info.

Explanation: The <a> tag is an inline element for hyperlinks.


Strong

Example:

This is important text.

Explanation: <strong> is an inline element used to give strong importance (bold by default).


Em

Example:

This is emphasized text.

Explanation: <em> is an inline element used for emphasis (italic by default).


Code

Example:

Use printf() to print in C.

Explanation: The <code> tag is inline, used for representing code snippets.


Image

Example:

Here’s an inline image:

Explanation: The <img> tag is inline and does not start on a new line.


Input

Example:

Enter name:

Explanation: <input> is an inline form control element.


Label

Example:

Explanation: <label> is inline and is used to define labels for form inputs.


Small

Example:

This is small text in a sentence.

Explanation: <small> is an inline element for side comments or fine print.


Abbr

Example:

WHO is an abbreviation.

Explanation: <abbr> is inline and provides the full form of abbreviations.
